Currency Transaction Report


Introduction:

A Currency Transaction Report (CTR) is a document that financial institutions in the United States are required to file with the Financial Crimes Enforcement Network (FinCEN) for certain transactions involving cash.


Financial institutions must file a CTR for any cash transaction over $10,000 in a single business day. This includes deposits, withdrawals, exchanges, or any combination of these transactions.


CTR reporting is part of the Bank Secrecy Act (BSA), which requires financial institutions to assist U.S. government agencies in detecting and preventing money laundering.


Who can file CTRs from the bank?

Generally, it will be the MLRO (Money Laundering Reporting Officer), however in some cases it can be filed by the Compliance officers as well.


Which jurisdiction's financial institution shoud report CTR?

It's applicable only in the United States.


To whom or which authority should the financial institution report the CTR?

Financial Crimes Enforcement Network (FinCEN)


What's the Threshold for CTR to be filed?

Any cash transaction over $10,000 in a single business day, including deposits, withdrawals, exchanges, or any combination of these transactions.


What constitute part of Currency Transaction Report form?


  • Financial Institution Information
    • Name of the Financial Insitution
    • Communication Address of the FI
    • SWIFT/BIC

  • Point of Contact - Financial Institution
    • Name
    • Contact Number
    • Email ID

  • Customer Information
    • Name of the customer
    • Communication address of the customer
    • Date of Birth
    • SSN Number
    • Occupation

  • Transaction Details
    • Date of the transaction
    • Amount
    • Account number
    • Type of Transaction

  • Additional Details
    • Multiple Transactions
    • Source of Funds
